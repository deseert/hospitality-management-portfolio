# hospitality-management-portfolio
phase2-C200-portfolio
Professional portfolio for Phase 2 of the course. This repository will collect artifacts, documents, and progress throughout the term.

## Problem Being Solved
 Many small restaurants struggle with inefficient inventory tracking, high labor costs, and manual scheduling, leading to waste, over/under-staffing, and lost revenue. This project aims to build a simple, integrated system to streamline operations.

## System Vision
The vision is to create an affordable, user-friendly enterprise resource planning (ERP) tool tailored for restaurant managers. It will integrate inventory management, staff scheduling, sales tracking, and basic reporting—built on or inspired by platforms like Odoo—to help owners reduce costs by 15-20% and improve team efficiency.

## My Role on the Team
 As the team lead and primary contributor, I handle requirements gathering, system vision documentation, initial architecture design, and mapping features to enterprise platforms.

Future additions will include:
- System architecture diagrams
- Odoo module mappings
- Requirements documents
- Any other project files
